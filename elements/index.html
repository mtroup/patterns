Task Buttons

HTML:

<button class="menu-button">Menu</button>

SASS SAMPLE

$btnMenu: #f0ad4e; .global-btn { @extend .btn; border-radius:0 !important; color: $white; &:hover { color: $white; @include backgroundHover(); } } .menu-button { @extend .global-btn; background: $btnMenu; }

NOTES:

All buttons are named by the function of the button first followed with "-button". Another example of this would be class="chat-button". There is no need to reference any CSS for styling results.

Search Form

HTML:

<form> <div class="input-group global-search"> <input class="form-control" placeholder="Search" type="text"> <span class="input-group-btn"> <button type="submit" class="search-button"><span class="glyphicon glyphicon-search"></span> </button> </span> </div> </form>

NOTES:

Reference search button with class="search-button" which in return will use the task button for search and apply the SASS variable's and mixins.

Submit Form

HTML:

<form> <div class="form-group"> <label>Sample Label</label> <input type="email" class="form-control" placeholder="Sample"> </div> <div class="form-group"> <label>Sample Label</label> <input type="email" class="form-control" placeholder="Sample"> </div> <button type="submit" class="submit-button">Submit</button> </form>

NOTES:

All border radius are removed with SASS - $border-radius-base: 0;
